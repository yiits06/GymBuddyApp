# GymBuddy 🏋️‍♂️

**Partnerini Bul, Limitlerini Zorla!**

## Getting Started
GymBuddy, spor salonunda kendine uygun antrenman partneri bulmanı, antrenmanlarını kaydetmeni ve çevrendeki diğer sporcularla etkileşime geçmeni sağlayan modern bir mobil uygulamadır. 

This project is a starting point for a Flutter application.
## 🚀 Özellikler

A few resources to get you started if this is your first Flutter project:
*   **Gelişmiş Eşleşme Sistemi:** Hedeflerine ve deneyim seviyene uygun sporcuları keşfet, beğen ve eşleş.
*   **Gerçek Zamanlı Mesajlaşma:** Eşleştiğin partnerlerinle anında iletişime geç.
*   **Anlık Bildirimler:** Yeni mesajlar ve eşleşmeler için Firebase altyapılı anlık bildirimler (Uygulama içi ve arka plan desteği).
*   **Antrenman Takibi:** Günlük antrenman programını, sürelerini ve ağırlıklarını kolayca kaydet.
*   **Yakındaki Sporcular:** Konum tabanlı harita entegrasyonu ile çevrendeki aktif sporcuları gör.
*   **Spor Salonu Detayları:** Salonun canlı kapasitesini ve aktif sporcuları gör, tek tıkla cihazın harita uygulamasıyla yol tarifi al.
*   **Çoklu Dil Desteği:** Türkçe ve İngilizce dil seçenekleri.
*   **Karanlık Tema (Dark Mode):** Göz yormayan, şık "Neon Lime" detaylı modern tasarım.

## 🛠️ Kullanılan Teknolojiler

samples, guidance on mobile development, and a full API reference.
*   **Mobil Framework:** [Flutter](https://flutter.dev/)
*   **State Management:** [Riverpod](https://riverpod.dev/) (`flutter_riverpod`)
*   **Backend & Veritabanı:** Supabase (Auth, PostgreSQL)
*   **Bildirimler:** Firebase Cloud Messaging (FCM) & `flutter_local_notifications`
*   **Harita & Konum:** `flutter_map`, `latlong2`, `geolocator`
*   **Çoklu Dil (i18n):** `easy_localization`
